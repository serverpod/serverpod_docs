# Serverpod Academy

[Serverpod Academy](https://academy.serverpod.dev) is a free video course on the Serverpod framework. Get Started gets you to a running server; the Academy is the deep dive.

The course is an introduction to professional backend development. You'll cover the concerns you'd hit shipping a real app: databases, security, caching, and unit testing.

**Ground Training**, the 4-hour course, takes you beyond Get Started into:

- Authentication
- Real-time streams
- File uploads
- Task scheduling
- Caching

You finish with a working app and a certificate.
