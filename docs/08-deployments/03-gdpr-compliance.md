# GDPR compliance

Serverpod Cloud projects are hosted in Frankfurt on Google Cloud Platform, which gives your project a strong foundation for GDPR compliance. Google LLC is an active participant in the EU-U.S. Data Privacy Framework.

Your data is stored in Frankfurt through Neon, which complies with the GDPR framework.

## Certifications

Serverpod has not yet been certified to ISO or SOC 2 standards. The underlying infrastructure providers that store and run your data are certified:

- Google Cloud Platform holds SOC 2, ISO/IEC 27001:2022, and ISO/IEC 27701:2019 certifications.
- Neon holds SOC 2, ISO/IEC 27001:2022, and ISO/IEC 27701:2019 certifications.

For the full list of certifications, see [Google Cloud compliance](https://cloud.google.com/compliance) and [Neon security](https://neon.com/security).
